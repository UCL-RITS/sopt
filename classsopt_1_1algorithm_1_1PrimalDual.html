Detailed Description

template<class SCALAR>
class sopt::algorithm::PrimalDual< SCALAR >

Primal Dual method.

This is a basic implementation of the primal dual method using a forward backwards algorithm. \(\min_{x, y, z} f(x) + l(y) + h(z)\) subject to \(Φx = y\), \(Ψ^Hx = z\) We are not implementing blocking or parallelism here.

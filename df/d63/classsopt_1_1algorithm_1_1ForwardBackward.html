Detailed Description

template<class SCALAR>
class sopt::algorithm::ForwardBackward< SCALAR >

Proximal Forward Backward algorithm.

\(\min_{x} γf(x) + \|y - Φx\|\) where \(y\) is a target vector.

Computes initial guess for x and the residual using the targets.

with y the vector of measurements

  • x = Φ^T y / ν
  • residuals = Φ x - y

This function simplifies creating overloads for operator() in FB wrappers.

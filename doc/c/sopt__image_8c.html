Read RGBA tiff image from file as greyscale image (the red channel of the RGBA image is read).

Note
Memory for img is allocated herein and must be freed by the calling routine.
Parameters
[out]imgGreyscale image read from tiff file.
[out]nxDimension of image in x direction.
[out]nyDimension of image in y direction.
[in]filenameName of file to read tiff image from.
[in]scaleScale factor to apply to image pixels (8 bit values read from tiff file are mapped to (val-offset)*scale.
[in]offsetOffset to apply to image pixels (8 bit values read from tiff file are mapped to (val-offset)*scale.

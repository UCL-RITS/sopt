Compute explicit representation of a sparse matrix.

Parameters
[out]AExplicit matrix representation of sparse matrix S. The matrix A is stored in column-major order, i.e. can be accessed as A[col_index * nrows + row_index].
[in]SSparse matrix to representation explicitly (passed by reference).
Note
Space for the output matrix A is allocated herein and must be freed by the calling routine.
